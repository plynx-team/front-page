Features

Workflow Editor

Interactive User Interface. You can clone successful experiment and reuse it or create one from scratch. PLynx manages history of the experiments and they can be reproduced. End-to-end workflows help preventing bias in data driven solutions.

Operations editor

Operations can be customized independently from the platform bt yhe users. They can be created dynamically for rapid prototyping, or using existing version control or CI/CD frameworks.

Monitor progress

Track the progress of the experiment. Each of intermediate operations produce results that you can inspect. Users can create multiple long-running experiments simultaniously and put them in the queue.
